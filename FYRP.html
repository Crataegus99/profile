RISC-V FPGA Project

Final Year Research Project

RISC-VFPGAProject

Overview

This project implements a RISC-V based embedded vision system on FPGA. It integrates camera input, VHDL processing modules, and real-time control.

System Architecture

The system uses NEORV32 soft-core processor integrated with custom image processing modules using BRAM and AXI interconnect.

Results

Achieved real-time object detection without GPU acceleration using optimized classical algorithms.
